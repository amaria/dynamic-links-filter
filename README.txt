INTRODUCTION
------------------

Provides a text filter that converts predefined keywords into links. The text
and urls for the links are managed via an administration page.

More information can be found on the project page.

INSTALLATION
-----------------

Install as you would any other module, see
https://drupal.org/documentation/install/modules-themes/modules-7 for further
information.
